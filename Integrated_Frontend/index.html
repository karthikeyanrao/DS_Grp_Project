About This Project

This integrated application demonstrates five different distributed systems communication models:

  • Socket Programming: Low-level TCP/IP communication for complaint management
  • Java RMI: Remote method invocation for distributed object access
  • RPC (REST API): Stateless HTTP-based communication for notice board
  • Peer-to-Peer: Decentralized file sharing without central server
  • Shared Memory: Inter-process communication with synchronization
